vue页面嵌入iframe,iframe页面接口请求不到

2023-03-22 166 字

前提： vue项目下嵌入iframe页面

问题： vue项目是https，嵌入iframe页面的src也是https，但是iframe页面里的请求是http，发送请求报错 Mixed Content: The page at ‘https://xxxxxx' was loaded over HTTPS, but requested an insecure form action ‘http://fxxxxxxx'. This request has been blocked; the content must be served over HTTPS.
意思大概就是https协议下的请求中有http请求 导致混合请求浏览器不允许。
解决方法：
https降级成http // 否掉 业务上指定不允许
vue里的pubilc中加入 // 没测试，不知道行不行
修改nginx代理请求

增加add_header Content-Security-Policy "upgrade-insecure-requests"

将http协议升级为https // 已经实现。可行
