html2canvas初使用

2023-04-03 403 字

需求是第一生产力，也是催动学习新技术的动力

阳光明媚的一天我接到了一个需求，要求弹框并且截图发送到邮箱。

vue2+element的框架

使用的html2canvas的框架，前人已经造好轮子了我们用就完了。

  1. 先安装 npm install –save html2canvas

官网 http://html2canvas.hertzen.com/

  1. html中的代码

    <template>
      <div>
        <el-dialog
          title="提示"
          :visible.sync="dialogVisible"
          width="30%"
          id="html2canvas"
          ref="html2canvas"
          >
          <span>这是一段信息</span>
          <span slot="footer" class="dialog-footer">
            <el-button @click="dialogVisible = false">取 消</el-button>
            <el-button type="primary" @click="saveImage()">生成图片</el-button>
          </span>
        </el-dialog>
      </div>
    </template>

  2. js中的代码

    import html2canvas from "html2canvas"


    saveImage() {
      const scrollTop = document.documentElement.scrollTop
      window.pageYoffset = 0
      document.documentElement.scrollTop = 0
      document.body.scrollTop = 0
      html2canvas(document.getElementById('html2canvas'), {
        backgroundColor: '#FFFFFF',
        useCORS: true,
        scale: 1
      }).then(canvas => {
        const imgData = canvas.toDataURL('image/png')
        const a = document.createElement('a')
        const event = new MouseEvent('click')
        a.href = imgData
        a.download = '测试'
        a.dispatchEvent(event)
        // 截完图后再把位置滚动到之前的位置
        document.documentElement.scrollTop = scrollTop
      })
    },

  3. 获取图片后转成base64然后再通过接口发送出去。




==注意事项==

因为我是要在ui插件上去截图。而不是在页面上截图 所以this.$ref[xxxx]的方式行不通，要用document.getElementById的方式获取元素。如果不是在ui上截图就可以用this.$ref[xxxx]的方式 例如

<div id='imgCanves' ref='imgCanves'>
  
</div>
